
# Python Database

This is Python Database Managemnt Project




## Tech Stack

**Server:** SQL, Python3.x


## Running Tests

Install

```bash
  pip install mysql-connector-python
```
Running No Argument

```bash
  Python3 database.py 
```

Running With Argument

```bash
  Python3 database.py host username password
```
Example
```bash
  Python3 database.py localhost root toor
```
